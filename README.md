# Security Posture Assessment for a Financial Institution

> A cybersecurity consulting case study demonstrating the planning, scoping, and documentation of a professional security assessment engagement.



## Overview

This repository showcases a cybersecurity consulting engagement conducted for a financial institution, **FinSecure**.

The objective of this project was to demonstrate the consulting process involved in planning a comprehensive security assessment after a phishing incident. Rather than focusing solely on technical testing, this project emphasizes client engagement, project planning, security documentation, compliance considerations, and business communication.

The primary deliverable is a professional **Statement of Work (SOW)**, which outlines the proposed engagement between the consulting firm and the client.



## Business Scenario

FinSecure is a mid-sized financial institution with approximately 500 employees and an expanding online presence.

Following a sophisticated phishing attack, several employee email accounts were compromised. Although no customer information was exposed, executive leadership recognized the need to evaluate the organization's overall cybersecurity posture.

The CEO engaged **Safempire Consulting** to perform a comprehensive security assessment designed to answer key business and security questions while minimizing disruption to normal operations.



## Business Objectives

The engagement was designed to address the following objectives:

* Assess the organization's current security posture.
* Evaluate the effectiveness of existing security controls.
* Identify weaknesses that could be exploited by ransomware, phishing attacks, or insider threats.
* Review alignment with PCI DSS and the Gramm-Leach-Bliley Act (GLBA).
* Produce practical recommendations for improving security maturity.
* Deliver a cost-effective engagement with clearly defined deliverables and timelines.
